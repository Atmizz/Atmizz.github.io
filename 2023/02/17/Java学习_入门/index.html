属性

属性和局部变量的不同
  1. 相同点：
    1. 定义变量格式： 数据类型 变量名=变量值
    2. 先声明，后使用。
    3. 变量都有对应的作用域
  2. 不同点：
    1. 声明位置不同
      1. 属性：直接定义在类里
      2. 局部变量：声明在方法内、方法形参、代码块内、构造器内、构造期内部
    2. 关于权限修饰符的不同
      1. 属性：可以声明属性时，指明其权限，使用权限修饰符（常见的权限修饰符：private、public、缺省、protected）。
      2. 局部变量：不可以使用权限修饰符。
    3. 默认初始化值的情况
      1. 属性：类的属性，根据其属性，都有默认初始化值。
      2. 局部变量：无默认初始化值。特别的，形参在调用时赋值即可。
    4. 在内存中加载的位置不同
      1. 属性：加载到堆空间。
      2. 局部变量：加载到栈空间中。
